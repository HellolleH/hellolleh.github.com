人工智能读书笔记(01) —— 人工智能概念

发表于 2017-10-01 |

引言

数千年来，人们一直试图理解人类是如何思考的 —— 仅仅少量的物质是怎样感知、理解、预测和操纵一个远大于自身，并且比自身复杂得多的世界？
人工智能（Artificial intelligence，简称 AI）是新兴的科学与工程领域之一。目前包含大量各种各样的子领域，范围从通用领域，如学习和感知，到专业领域，如下棋、证明数学定理、写诗、自动驾驶、诊断疾病等。

什么是人工智能？

  • 像人一样的思考，并且合理地思考
  • 像人一样的行动，并且合理地行动

由阿兰·图灵（Alan Turing）提出的图灵测试（Turing Test）的目的就是为人工智能提供一个令人满意的定义。

如果一个人类询问者，在提出一些书面问题后，不能区分书面回答来自人还是计算机，那么这台计算机就通过测试。

在现如今，要想通过编程，让计算机通过测试，计算机还需要具备以下能力：

  1. 自然语言能力 —— 能够用语言交流
  2. 知识表示能力 —— 能够存储知道或听到的信息
  3. 自动推理能力 —— 运用存储的信息回答问题并推出新的结论
  4. 机器学习能力 —— 能够适应新情况并具有检测和预测能力
  5. 计算机视觉 —— 能够感知物体
  6. 机器人学 —— 能够操纵或移动物体

Agent

Agent 就是能够行动的某种东西（源自于拉丁语 agere，意为“去做”）。

当然，所有的计算机程序都能够做某些事情。但是，在人工智能领域，我们期望 Agent 能够做更多的事情，诸如：自主操作、感知环境、长期持续、适应变化以及创建和追求目标等。

合理 Agent 是为了实现 最佳结果，或者，当存在不确定性时，为了实现最佳期望结果而行动的 Agent。

不过，在复杂环境中，要实现完美的合理性，计算要求太高。后续会介绍 有限合理性 —— 当没有足够的时间来完成所有想做的计算时，仍然能够恰当的行动。

一句话小结

简单明确了 人工智能 的概念，以及要实现人工智能 计算机还需具备的能力，另外还明确了 Agent 概念。
